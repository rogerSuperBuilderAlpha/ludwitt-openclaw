## What is Ludwitt?

Ludwitt is an **AI-powered learning platform** with two major products:

1. **Basics** — Adaptive K-12 + adult literacy/numeracy learning (Math, Reading, Logic, Latin, Greek, Writing) addressing the global skills crisis documented by PIAAC
2. **Developer Training (ALC)** — Learn to build with AI coding tools (Cursor, Claude Code, OpenClaw) through guided projects, deployment, and open-source contributions

## The Problem We Solve

**PIAAC 2023 data reveals an alarming crisis:**

- US adult literacy scores dropped 12 points (271→258) from 2017-2023
- 28% of US adults now struggle with basic literacy (up from 19%)
- 34% of US adults struggle with basic numeracy (up from 29%)
- Approximately 54 million Americans cannot read at a functional level
- Approximately 66 million Americans struggle with basic math

This affects employment, healthcare navigation, financial literacy, and civic participation.

## Our Solution

**Adaptive Learning Engine**: Problems adjust in real-time to maintain 70-85% success rate—the optimal challenge zone based on Vygotsky's Zone of Proximal Development.

**Subjects Covered**:

- Mathematics (arithmetic through algebra)
- Reading comprehension and vocabulary
- Logic and critical thinking
- Latin and Greek (classical languages)
- Writing skills

**Key Features**:

- AI tutoring powered by Anthropic Claude
- Spaced repetition for long-term retention
- Gamification (XP, streaks, achievements, leaderboards)
- Privacy-first design—adults can practice without stigma
- Mobile-responsive for learning anywhere

## Learning Science Foundations

Built on proven research:

- **Zone of Proximal Development** (Vygotsky) - Optimal challenge level
- **Spaced Repetition** (Ebbinghaus) - Long-term memory retention
- **Mastery Learning** (Bloom) - Competence before advancement
- **Immediate Feedback** (Hattie) - High-impact intervention
- **Growth Mindset** (Dweck) - Effort over innate ability

## Developer Training (ALC)

Ludwitt's developer training track teaches complete beginners to build real projects using AI coding tools. Three paths are available:

- **Cursor path**: Install Cursor AI editor, build a personal website, deploy to Vercel
- **Claude Code path**: Install Claude Code CLI, build a tweet automation agent, deploy to Firebase
- **OpenClaw path**: Install OpenClaw agent platform, build website + email, deploy

Students follow a 19-step journey: Git/GitHub setup, tool-specific project building, deployment, open-source contribution (PR to cursorboston.com), Loom showcase videos, 5-year vision document, and AI-generated custom projects.

**Skills taught**: Git, GitHub, web development, AI-assisted coding, deployment (Vercel/Firebase), open-source contributions, portfolio building.
