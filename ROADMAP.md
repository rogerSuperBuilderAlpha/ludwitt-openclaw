# Roadmap

Development priorities for Ludwitt. This roadmap reflects current plans and is subject to change based on community feedback.

## Current Focus — v1.1

- [ ] Expand test coverage to 30%+ (component tests, API route tests, E2E specs)
- [ ] Add screenshots and demo content to README
- [ ] Automated releases via release-please
- [ ] Codecov integration with coverage badge in README
- [ ] Resolve `--legacy-peer-deps` requirement

## Near-Term — v1.2

- [ ] Basics: expand problem sets for Latin, Greek, and Logic
- [ ] Basics: offline mode for problem solving (PWA enhancements)
- [ ] ALC: project template gallery for portfolio projects
- [ ] University: peer review matching algorithm improvements
- [ ] Performance: code-split large components (ProgressionGatePage, NotificationBell)

## Medium-Term — v2.0

- [ ] Multi-tenant support for schools and organizations
- [ ] Admin dashboard for self-hosted instance management
- [ ] Student analytics dashboard with learning insights
- [ ] Mobile-responsive redesign for Basics practice mode
- [ ] Internationalization (i18n) framework

## Future Considerations

- Native mobile app (React Native or PWA improvements)
- LMS integration (LTI protocol support)
- Plugin system for custom learning modules
- Kubernetes deployment manifests
- Multi-architecture Docker builds (ARM64/AMD64)
